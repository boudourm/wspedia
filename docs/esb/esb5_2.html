Routage

Le routage représente une caractéristique essentielle d'un ESB. Il s’agit d’utiliser un ensemble de règles métiers dans l’ESB pour acheminer les messages du fournisseur au consommateur et vice-versa ; c’est la capacité de décider la destination d'un message lors de son transport.

×

Le routage dans l’ESB est dit dynamique lorsque le routeur est capable de reconfigurer les règles de routage sur la base des messages de configuration qui peuvent être envoyés par les destinations participantes (les services).

La plupart des ESB utilisent le système JMS (Java Messaging Service) pour router les messages vers différents composants.
